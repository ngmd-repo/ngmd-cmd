# Command operations package for @ngmd libraries

## Description

The functionality of this library is used in:

- CI/CD processes for [@ngmd/translate](https://github.com/Octavian69/ngmd-translate) library

## Installation

```bash
yarn add @ngmd/cmd
# or
npm i @ngmd/cmd
```
